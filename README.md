Tattoo Studio Website

A modern, responsive tattoo studio website built with a focus on clean design, performance, and scalability. This project showcases a professional frontend architecture suitable for real-world production use.

Live Demo

(https://www.squaretattoo.com/)

Project Overview

This website is designed for a tattoo studio to present its brand, artwork, and services through a visually engaging and user-friendly interface. The layout and components are structured to allow easy customization and future feature expansion.

Getting Started
git clone <YOUR_GIT_URL>
cd <PROJECT_NAME>
npm install
npm run dev


The development server will start on the default Vite port.
